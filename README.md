# QR Code Generator

A simple web-based QR code generator. This project allows users to generate QR codes directly in their browser using a user-friendly interface.

## Features

- Generate QR codes from any text or URL
- Download the generated QR code as an image
- No backend required; works entirely in the browser

## Getting Started

1. Clone this repository:

   ```sh
   git clone https://github.com/your-username/qr-code-generator.git
   ```

2. Open `qr_generator.html` in your web browser.

## Usage

- Enter the text or URL you want to encode.
- Click the "Generate" button to create a QR code.
- Download or screenshot the QR code as needed.

## Dependencies

- [qrcode.js](https://github.com/davidshimjs/qrcodejs) or similar (add to your HTML if not already included)
